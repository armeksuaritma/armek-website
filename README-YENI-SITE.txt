ARMEK YENİ KURUMSAL WEB SİTESİ

GitHub'a yüklerken bu klasörün İÇİNDEKİ dosyaları deponun ana dizinine koyun.
index.html doğrudan ana dizinde görünmelidir.

GitHub Pages kurulumu:
1) GitHub deposu > Settings > Pages
2) Source: Deploy from a branch
3) Branch: main, klasör: /(root)
4) Save

Not: /admin panelinin GitHub üzerinde içerik kaydedebilmesi için ayrıca GitHub OAuth kurulumu gerekir.
Site ve WhatsApp/telefon bağlantıları yönetici panelinden bağımsız olarak çalışır.
