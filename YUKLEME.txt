ARMEK GENİŞ YÖNETİM PANELİ GÜNCELLEMESİ

Bu pakette 7 dosya vardır. ZIP'i açın ve içindeki dosya/klasör yapısını bozmadan GitHub ana dizinine yükleyin.

Değişecek dosyalar:
- app.js
- index.html
- styles.css
- admin/config.yml

Yeni dosyalar:
- site-control.json
- categories.json

Yükleme:
1. GitHub deposunda Add file > Upload files seçin.
2. ZIP'i değil, ZIP'ten çıkardığınız dosyaları ve admin klasörünü sürükleyin.
3. Commit changes düğmesine basın.
4. Yayın tamamlanınca /admin/ panelinde Site Kontrol Merkezi ve Kategori Yönetimi görünecektir.
