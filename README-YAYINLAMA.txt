ARMEK YENİ WEB SİTESİ — YAYINLAMA

1. ZIP dosyasını açın.
2. İçindeki tüm dosya ve klasörleri mevcut GitHub deposunun ana dizinine yükleyin.
3. Eski dosyaların üzerine yazılmasına izin verin ve değişiklikleri main dalına kaydedin (commit).
4. Netlify mevcut GitHub deposuna bağlıysa site otomatik olarak yeniden yayınlanır.
5. Yönetici paneli: https://armeksuaritma.com.tr/admin/

ÖNEMLİ:
- Site logosu, ana sayfa sağ görseli, hakkımızda görseli ve kampanya görseli panelde ayrı alanlardır.
- GitHub deposu veya Netlify sitesi değiştirilmediği sürece alan adı/DNS ayarı gerekmez.
- Yayından sonra tarayıcıda eski sürüm görünürse Ctrl+F5 ile zorla yenileyin.
